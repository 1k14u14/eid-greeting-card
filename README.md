# 🌙 Cinematic Eid Greeting Card

A responsive, interactive web application designed to deliver a cinematic Eid Mubarak greeting. This project features a timed visual sequence, synchronized background audio, and an integrated digital "THR" (payment) request utilizing DANA.

**[🔴 Live Demo: View the Greeting Card Here](https://1k14u14.github.io/eid-greeting-card/)** 

---

## ✨ Features

* **Cinematic UI Sequence:** Smooth, CSS-driven fade-ins and transitions orchestrated by JavaScript timeouts to create a video-like presentation.
* **Audio Integration:** Background music player with a floating, user-controlled mute/unmute toggle.
* **Interactive DOM Manipulation:** A start screen to bypass browser autoplay restrictions, ensuring a seamless audio-visual experience.
* **Payment Gateway Linking:** A built-in "Bagi THR" feature that links directly to a DANA request URL for seamless digital gifting.
* **Custom SVG/Image Assets:** Beautifully glowing visuals, including a festive Ketupat design.

## 🛠️ Tech Stack

This is a lightweight, dependency-free frontend project utilizing:
* **HTML5:** Semantic structure and audio handling.
* **CSS3:** Flexbox, radial gradients, keyframe animations, and filter dropshadows.
* **Vanilla JavaScript:** DOM manipulation, timeout sequencing, and audio state management.

## 🚀 How to Run Locally

If you want to download and modify this project:

1. Clone the repository:
   ```bash
   git clone https://github.com/1k14u14/eid-greeting-card.git
   ```
2. Navigate to the project directory:

```bash
cd eid-greeting-card
```
3. Open index.html in your browser.
   (Note: For the audio functionality to work perfectly without browser restrictions, it is recommended to open the project using a local server like the VS Code Live Server extension).
